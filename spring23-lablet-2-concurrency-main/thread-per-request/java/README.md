## Setup

First, install the Java sdk. On Ubuntu:

```
sudo apt install default-jdk
```

Next, compile the client and server using the `javac` command:

```
javac Client.java Server.java
```

Run the server first using the `java` command:

```
java Server
```

Finally, run the client:

```
java Client
```

More information can be found in the following tutorial: https://www.geeksforgeeks.org/multithreaded-servers-in-java/
